Coreshift Kanban Engine

Build once. Clone for every project. Stays in sync as part of the work.

The shared rendering engine + template kit that turns project-by-project boards into one system. Built first for Sentinel; every Coreshift project that follows gets the same board structure for free. This board tracks improvements to the tool itself.

Current phase Phase 1 · Engine + template shipped
Next milestone Operator demo Before retrofitting other boards

The Three Goals

🧰
Build once, clone everywhere
The engine is reusable. Adding a project is three files copied from tools/template — no surgery on the renderer.
📝
Markdown is the source of truth
Cards live in KANBAN.md (version-controlled), not in rendered HTML. Edit the file, run the build, push.
⚡
Stays in sync as part of the work
The kanban refresh is part of every ship — not a separate doc pass. If the board lags the code, the convention is broken.
